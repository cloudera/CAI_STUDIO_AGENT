# Release Notes

## September 30th, 2025

[Detailed Release Notes](./release_notes/2025_09_30/release.md)

* Added support for Bedrock LLMs.
* The Ops & Metrics server is no longer a separate service and is now embedded in the main Agent Studio application.
* Each deployed workflow now includes its own dedicated Ops & Metrics server.
* Agent Studio now supports air-gapped environments.
* Various UI fixes and improvements.
